Starting point

Bare ground first, then client drawings and prepared lumber

The first photos in the album are the true before condition: only the fenced dirt area was there, with no planter or bench installed yet.

The client provided the drawings, instructions, and prepared lumber. One technician matched the boards to the plan, set supports, fastened the walls, and corrected pieces that had been labeled or prepared incorrectly.

Scope of work

  • Document the bare-ground before condition
  • Review client drawings and supplied instructions
  • Sort labeled and prepared boards by section
  • Set support posts for the planter structure
  • Attach boards and assemble the flower bed walls
  • Rework boards where labels or prep did not match the plan
  • Install the built-in bench last
